Misión Inicial

Fase 1 · El archivo de imágenes

Recupera el archivo visual de la empresa

Objetivo de la fase: Analizar formatos de imagen, realizar una adquisición con periféricos y decidir la resolución adecuada según la finalidad.

Criterios trabajados

  • a) Formatos de imágenes
  • b) Adquisición con periféricos
  • c) Resoluciones según finalidad

Conceptos clave

  • Mapa de bits frente a imagen vectorial.
  • Formatos habituales: JPG, PNG, GIF, SVG, TIFF y RAW.
  • Resolución, dimensiones en píxeles y peso del archivo.
  • Captura mediante escáner, cámara o móvil.
  • Diferencias entre uso para web, documento y cartel impreso.

Caso práctico · Clasificación del banco de imágenes corporativo

Contexto: La empresa ha reunido imágenes para preparar una campaña interna. Antes de diseñar documentos y publicaciones, debes ordenar los archivos, identificar formatos y decidir cuáles sirven para web, informe o impresión.

Material necesario

  • Ordenador con visor o editor de imágenes
  • Móvil con cámara o escáner
  • Carpeta de trabajo RA6_Fase1
  • Documento o libreta para registrar datos

Guía detallada de realización

Paso 1. Prepara la estructura de trabajo

  1. Crea una carpeta llamada RA6_Fase1.
  2. Dentro crea tres subcarpetas: Originales, Capturas e Informe.
  3. Comprueba que todo el trabajo de esta fase se guardará dentro de esa carpeta.

Comprobación esperada: La carpeta RA6_Fase1 contiene exactamente las tres subcarpetas indicadas.

Paso 2. Reúne cuatro imágenes distintas

  1. Consigue una fotografía en JPG, un logotipo PNG con transparencia, una imagen vectorial en SVG o PDF exportado desde vectorial y una imagen de alta calidad en TIFF o similar si está disponible.
  2. Guarda todos los archivos en la carpeta Originales.
  3. Renombra los archivos con nombres claros: foto_evento, logo_empresa, icono_vectorial e imagen_calidad.

Comprobación esperada: Hay al menos cuatro archivos distintos y bien identificados.

Paso 3. Crea una tabla de análisis

  1. Abre un documento de texto o una hoja de cálculo.
  2. Haz una tabla con las columnas: Archivo, Formato, Peso, Dimensiones, Uso recomendado.
  3. Rellena una fila por cada imagen reunida.

Comprobación esperada: La tabla compara cuatro archivos e indica un uso recomendado por cada uno.

Paso 4. Realiza una adquisición propia

  1. Escanea un documento sencillo o fotografía con el móvil una tarjeta, cartel o portada.
  2. Pasa la imagen al ordenador.
  3. Guárdala en la carpeta Capturas con el nombre captura_empresa_01.

Comprobación esperada: Existe una imagen nueva obtenida mediante periférico.

Paso 5. Compara resolución y finalidad

  1. Revisa el tamaño o resolución de cada archivo.
  2. Escribe una breve explicación de 4 a 6 líneas indicando qué archivo usarías para web, cuál para documento y cuál para impresión.
  3. Guarda esa explicación en la carpeta Informe.

Comprobación esperada: Hay una conclusión razonada sobre formato, resolución y uso final.

Paso 6. Prepara la entrega

  1. Haz una captura de pantalla de la carpeta RA6_Fase1 abierta.
  2. Comprueba que incluye originales, captura propia y explicación final.
  3. Sube la práctica al entorno indicado por el profesorado.

Comprobación esperada: La entrega queda completa y preparada para revisión docente.

Producto o entrega esperada

Carpeta organizada con imágenes reales, una adquisición propia y una tabla comparativa con conclusiones técnicas.
