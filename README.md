# RA6 · Aventura Multimedia

Web estática lista para GitHub Pages.

## Incluye
- 6 fases tipo aventura
- casos prácticos guiados paso a paso
- guardado del nombre del alumno
- progreso persistente en el navegador
- validación docente con código diario oculto en campo password
- código de verificación con formato `mmddaa`
- test de 10 preguntas por fase
- superación con 8/10 o más
- hasta 3 intentos
- autocorrección final si no supera tras 3 intentos
- certificado final con valoración por fases

## Código docente
No es visible en la interfaz. Se calcula automáticamente en `script.js` con la fecha del día usando el formato `mmddaa`.

Ejemplo:
14 de abril de 2026 -> `041426`
