# RA6 · Aventura Multimedia (Modo videojuego)

Proyecto listo para subir a un repositorio y publicar en GitHub Pages.

## Incluye
- 6 fases con estética de misión / videojuego
- prácticas guiadas paso a paso
- progreso persistente con `localStorage`
- nombre del estudiante guardado
- validación docente por código oculto en campo de contraseña
- código diario con formato `mmddaa`
- test de 10 preguntas por fase
- superación con 8/10 o más
- hasta 2 intentos
- avance de fase tras 2 intentos con registro de resultado en certificado
- certificado final con valoración por fases
- fase 4 con banco de recursos educativos públicos sobre vídeo y audio digital

## Código docente
El sistema usa automáticamente la fecha del día en formato:
`mmddaa`

Ejemplo:
14 de abril de 2026 -> `041426`

## Acceso docente directo
Si se guarda el nombre de jugador `tmilgar`, se activa el modo docente y se puede revisar cualquier fase sin validación ni test.

## Subida al repo
Sube todos los archivos a la raíz del repositorio y publica con GitHub Pages.
