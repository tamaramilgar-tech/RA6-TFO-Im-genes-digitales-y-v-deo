Guía detallada en GIMP
  1. Abre GIMP y pulsa Archivo → Nuevo. Escribe 1080 × 1350 px.
  2. Ve a Archivo → Abrir como capas y selecciona la imagen principal.
  3. Usa Herramienta Escalar para ajustar tamaño y Herramienta Mover para recolocar.
  4. Para el nombre de la marca, pulsa la Herramienta Texto y escribe arriba.
  5. Añade un eslogan y una llamada a la acción con la misma herramienta.
  6. Si quieres fondo de color, usa Capa → Nueva capa y luego Cubo de relleno.
  7. Guarda el editable con Archivo → Guardar como y entrega con Archivo → Exportar como.
Guía detallada en Photopea
  1. Abre Photopea y pulsa New Project. Escribe 1080 × 1350.
  2. Importa la imagen con File → Open & Place.
  3. Usa Edit → Free Transform para ajustarla.
  4. Con la herramienta T, añade nombre de marca y eslogan.
  5. Ordena elementos desde Layers.
  6. Guarda editable con File → Save as PSD.
  7. Entrega final con File → Export As → JPG o PNG.
