Misión de Diseño

Fase 3 · El estudio creativo

Compón una imagen corporativa lista para insertar en documentos

Objetivo de la fase: Trabajar con selecciones, capas, texto y ajustes básicos para integrar imágenes digitales en documentos de empresa.

Criterios trabajados

  • Manipulación de selecciones, máscaras y capas
  • Inserción de textos y ajustes de imagen y color

Conceptos clave

  • Capas y orden de apilado.
  • Recorte y encuadre.
  • Brillo, contraste y color.
  • Selecciones básicas y máscaras sencillas.
  • Inserción de texto y jerarquía visual.

Caso práctico · Crear un cartel interno para la empresa

Contexto: El departamento de comunicación interna necesita un cartel para anunciar una campaña de reciclaje. Debes usar una fotografía de fondo, un logotipo y texto legible, aplicando un proceso de edición ordenado.

Material necesario

  • Editor con capas
  • Fotografía de fondo
  • Logotipo PNG
  • Carpeta RA6_Fase3

Guía detallada de realización

Paso 1. Crea el proyecto editable

  1. Abre la fotografía elegida.
  2. Guarda una copia editable en la carpeta RA6_Fase3 antes de empezar a modificarla.
  3. Decide si tu cartel será vertical o cuadrado.

Comprobación esperada: Existe un archivo editable base del cartel.

Paso 2. Ajusta el encuadre

  1. Recorta la imagen para mejorar la composición.
  2. Deja espacio suficiente para colocar títulos.
  3. Evita eliminar el elemento principal de la fotografía.

Comprobación esperada: La imagen queda equilibrada y preparada para recibir texto.

Paso 3. Mejora el fondo

  1. Aplica ajustes suaves de brillo, contraste o saturación.
  2. Evita filtros exagerados.
  3. Guarda los cambios sin perder el archivo editable.

Comprobación esperada: El fondo tiene mejor legibilidad y apariencia profesional.

Paso 4. Inserta una capa de apoyo

  1. Añade una forma o rectángulo semitransparente detrás del texto.
  2. Coloca esa capa en la zona donde escribirás el mensaje.
  3. Ajusta opacidad hasta que el texto se lea bien.

Comprobación esperada: La composición facilita la lectura del mensaje principal.

Paso 5. Añade logotipo y texto

  1. Importa el logotipo en una capa superior y colócalo en una esquina.
  2. Escribe un título y un subtítulo, por ejemplo Semana Verde y Reduce, reutiliza y recicla.
  3. Alinea visualmente los elementos para que el diseño sea limpio.

Comprobación esperada: El cartel comunica con claridad y mantiene coherencia visual.

Paso 6. Usa una selección o máscara

  1. Selecciona una zona concreta para desenfocar, aclarar o destacar un elemento.
  2. Si tu programa lo permite, aplica una máscara sencilla para suavizar una zona.
  3. Guarda una versión editable y otra exportada.

Comprobación esperada: Demuestras el uso de selección o máscara dentro del diseño.

Paso 7. Exporta y documenta

  1. Exporta una versión final en PNG o JPG.
  2. Haz una captura del panel de capas si tu programa lo muestra.
  3. Escribe tres líneas explicando qué ajustes has aplicado.

Comprobación esperada: La entrega incluye diseño final, archivo editable y breve explicación técnica.

Producto o entrega esperada

Cartel final con capas, texto y ajustes, acompañado de una breve explicación del proceso.
