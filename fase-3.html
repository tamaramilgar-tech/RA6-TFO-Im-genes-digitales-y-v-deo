Misión 3

Fase 3 · Estudio creativo

Diseña un cartel interno como si fuera un encargo real del departamento de comunicación.

Conceptos clave

  • Capas y orden de apilado.
  • Recorte y composición.
  • Brillo, contraste y saturación.
  • Selecciones básicas.
  • Inserción de texto y logotipo.

Caso práctico · Crear un cartel interno para la empresa

Contexto: Debes crear un cartel para anunciar la campaña “Semana Verde” usando una imagen de fondo, un logotipo y texto legible.

Programa recomendado

Opción recomendada

Photopea, gratuito y online. Muy adecuado para trabajar con capas, texto y exportación.

Opción instalada

GIMP, gratuito y potente para edición por capas.

Consejo

Si el alumnado no tiene editor instalado, Photopea suele ser la opción más directa.

Material necesario

  • Photopea o GIMP
  • Fotografía de fondo
  • Logotipo PNG
  • Carpeta RA6_Fase3

Guía detallada de realización

Paso 1. Preparar carpeta y archivos

  1. Crea RA6_Fase3.
  2. Guarda dentro una foto de fondo y un logotipo PNG.
  3. Comprueba que ambos archivos se abren correctamente.

Comprobación esperada: La carpeta contiene los recursos necesarios.

Paso 2. Crear el lienzo

  1. Abre Photopea o GIMP.
  2. Crea un archivo nuevo de 1080 x 1350 píxeles.
  3. Guarda una primera versión editable del cartel.

Comprobación esperada: Existe un lienzo vertical preparado.

Paso 3. Insertar fondo

  1. Importa la fotografía como capa.
  2. Escálala para que cubra todo el lienzo.
  3. Centra la parte más importante de la imagen.

Comprobación esperada: La fotografía ocupa el fondo completo.

Paso 4. Mejorar legibilidad

  1. Ajusta brillo y contraste ligeramente.
  2. Si hace falta, reduce un poco la saturación.
  3. Evita cambios exagerados.

Comprobación esperada: El fondo queda equilibrado y más legible.

Paso 5. Crear zona de texto

  1. Inserta una nueva capa.
  2. Dibuja un rectángulo oscuro detrás de la zona del texto.
  3. Baja la opacidad al 50–70%.

Comprobación esperada: Se crea una banda semitransparente para el texto.

Paso 6. Añadir título y subtítulo

  1. Escribe SEMANA VERDE.
  2. Añade debajo Reduce, reutiliza y recicla.
  3. Usa color claro y tamaño suficiente.

Comprobación esperada: El mensaje principal se lee con claridad.

Paso 7. Añadir logotipo y selección

  1. Importa el logotipo en una esquina.
  2. Usa una selección para destacar o desenfocar una zona concreta.
  3. Guarda una versión editable y otra final.

Comprobación esperada: El cartel demuestra uso de capas, texto y selección.

Producto o entrega esperada

Cartel final, archivo editable y una breve explicación del proceso.
